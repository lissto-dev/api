# What is Lissto
  
Lissto is a DevEnv and DevEx(perience) platform that simplifies the development 
of applications on Kubernetes.
It bridges the gap between Compose loved by developers and Kubernetes 
loved by DevOps and platform engineers.  

# Lissto API

The API component of the Lissto platform.

## Quick Start

Build:
```bash
make build
```

Run:
```bash
make run
```
